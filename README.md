# Economic-and-Decision-Analysis---Purcari-Wineries-Group
Proiectul reprezintă o soluție software integrată pentru gestionarea și analizarea datelor economice ale grupului Purcari Wineries. Obiectivul principal a fost examinarea performanței celor 4 branduri principale și optimizarea procesului decizional prin tehnici avansate de modelare a datelor.
Instrumente utilizate:
- Microsoft Access: Proiectarea bazei de date relaționale și implementarea interogărilor complexe.
- Microsoft Excel:
  - Data Quality: Curățarea și validarea datelor;
  - Analiză descriptivă: Utilizarea Data Analysis Toolpak pentru indicatori statistici.
- Power BI: Crearea unui dashboard interactiv pentru monitorizarea indicatorilor cheie (KPIs), evoluția vânzărilor pe ani și distribuția volumului de acțiuni tranzacționate.

Funcționalități cheie:
- Analiza pieței de capital: Monitorizarea acțiunilor Purcari în perioada 2018-2023, identificând trenduri și momente de volatilitate.
- Previziuni: Analiza volumului lunar de tranzacționare și identificarea strategiilor optime pentru maximizarea profitului total.
- Vizualizare avansată: Grafice de tip Line Chart, Stacked Column și Pie Chart pentru analiza comparativă a brandurilor.
